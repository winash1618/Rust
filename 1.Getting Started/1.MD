# Getting started
## Intstall
```
curl --proto '=https' --tlsv1.3 https://sh.rustup.rs -sSf | sh
```
## Updating
```
rustup update
```
## Uninstall
```
rustup self uninstall
```
## Writing and Running a Rust Program
- create a rust file in the following format Filename: main.rs
```
$ rustc main.rs
$ ./main
Hello, world!
```
